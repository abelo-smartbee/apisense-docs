# Protocol 3 — brood comb section with affected brood

!!! warning "Only after prior contact"
    Take the brood-comb section **only** if:

    - the beekeeper has detected concerning changes in the brood, **or**
    - a **"foulbrood"** alert has appeared in the Apisense app.

    **Contact Dr Maciej Bryś first** (tel. +48 455 501 920, [maciej.brys@apisense.ai](mailto:maciej.brys@apisense.ai)) to agree on the details.

## Materials

When collecting samples from **two colonies**, prepare:

- **2 pairs of disposable gloves** (a fresh pair for each colony)
- **2 paper envelopes**
- **2 blades**, **or** one blade and a way to **flame-sterilize** it between colonies

## Procedure

1. Put on a **fresh pair of gloves** for each colony.
2. Cut a **brood-comb section with affected brood, ~10×10 cm**.
3. Place the section into a **paper envelope**.
4. If you use the same blade for several colonies — **flame-sterilize it** between colonies.

## Register the sample in the app

1. In the Apisense app: ***add → register sample → collection date → sample type = brood comb***.
2. The app generates a **unique test code**.
3. **Write the code on the envelope** with a permanent marker.
