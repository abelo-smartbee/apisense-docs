# Próbki — program „Zdrowotność rodzin pszczelich, Wiosna 2026"

Sezonowy program monitoringu zdrowia pszczół prowadzony przez Apisense we współpracy z Uniwersytetem Przyrodniczym w Lublinie. Strona zawiera kompletny zestaw protokołów pobierania próbek oraz zalecenia dotyczące przeglądu rodziny.

!!! warning "Przeczytaj przed przystąpieniem"
    Przed pobraniem próbek, określeniem stopnia porażenia *Varroa destructor* i wykonaniem przeglądów **zapoznaj się z całością niniejszej instrukcji**.

## Protokoły

| | Protokół | Status |
|---|---|---|
| 1 | [Monitoring *Varroa destructor*](protocol-1-varroa.md) — flotacja cukrem pudrem + osyp na wkładce | Obligatoryjny |
| 2 | [Pobranie żywych pszczół](protocol-2-live-bees.md) — ~40 robotnic ze środka gniazda | Obligatoryjny |
| 3 | [Wycinek plastra](protocol-3-brood-comb.md) — 10×10 cm ze zmienionym czerwiem | Warunkowy (po kontakcie) |
| 4 | [Martwe pszczoły](protocol-4-dead-bees.md) — osyp z dennicy | Zalecany |

Dodatkowo: [Przegląd rodziny pszczelej](colony-inspection.md) — wymagane zdjęcia ramek i lista obserwacji do odnotowania w aplikacji Apisense.

## Wysyłka do badań

1. Opisane próbki (z naniesionymi kodami badań z aplikacji) umieść razem w **kartonowym pudełku**.
2. Nadaj paczkę **w dni robocze (poniedziałek–czwartek)**, aby uniknąć przechowywania przesyłki przez weekend.
3. Wyślij **pocztą lub kurierem 24 h**, aby próbki jak najszybciej dotarły do laboratorium.
